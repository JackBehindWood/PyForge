# PyForge

**A modular build system manager in Python.**  
Currently supports Premake and is designed for easy extension to other build systems like CMake. Fully integrates with Git, including support for submodules.

## Features
- Modular architecture for supporting multiple build systems
- Premake project support out-of-the-box
- Git integration (submodules, commits, updates)
- Easily extensible for future build systems

## Installation

Clone the repository and install in editable mode:

```bash
git clone https://github.com/yourusername/PyForge.git
cd PyForge
pip install -e .
```
